# Senior Engineer Skill

This skill provides expert software engineering guidance for project maintenance, incremental modernization, and production quality optimization.

## Overview
The Senior Engineer Skill helps analyze projects, validate configurations, update dependencies, optimize code, and maintain clear traceability. It follows industry best practices for software maintenance and modernization.

## Features
- Bilingual support (English/ Español)
- Structured analysis and recommendations
- Focus on security, stability, and maintainability
- Clear traceability of suggested changes
- Minimal risk, reversible improvements

## Usage
Activate this skill when you need expert guidance on:
- Software project maintenance
- Legacy system modernization
- Codebase optimization
- Development practice establishment

The skill will prompt you to select a language and provide project details, then return structured guidance in your chosen language.

## Structure
- `SKILL.md`: Detailed skill description, activation criteria, instructions, and examples
- `README.md`: This file

## Language Support
This skill automatically responds in the language selected by the user (English or Español).
